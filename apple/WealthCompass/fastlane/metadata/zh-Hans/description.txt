Wealth Compass 是一款私人、本地优先的个人财务仪表板，帮助您在一个地方了解净资产、现金流、投资、加密资产和负债。

建立清晰的财务图景：

- 一目了然地查看净资产、现金余额、投资和加密货币
- 按类别和可视化趋势跟踪收入和支出
- 安排定期交易并接收可选的本地提醒
- 监控股票、ETF 和加密货币持仓
- 查看成本基础、盈亏和投资组合分配
- 与资产一起跟踪负债
- 通过自动本地快照关注净资产历史
- 使用缓存的欧洲央行参考汇率进行价值转换
- 使用隐私模式立即隐藏敏感金额
- 用 Face ID 或 Touch ID 保护应用
- 随时导出、导入和删除本地数据

您的财务记录保留在设备上。Wealth Compass 不需要账户，也不在开发者运营的服务器上存储您的投资组合或交易历史。

可选的市场价格更新使用您提供的 API 凭证。凭证安全存储在 iOS 钥匙串中。

Wealth Compass 是一款信息跟踪工具。它不连接金融机构、不执行交易、不提供财务建议，也不保证第三方市场数据的准确性。
