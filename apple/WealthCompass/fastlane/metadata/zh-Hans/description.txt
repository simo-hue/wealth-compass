Wealth Compass 是一款私人、本地优先的个人财务仪表盘，帮助您在一个地方了解净资产、现金流、投资、加密资产和负债情况。

构建清晰的财务状况图：

- 一目了然地查看净资产、现金余额、投资和加密货币
- 使用类别和可视化趋势跟踪收入和支出
- 安排定期交易并接收可选的本地提醒
- 监控股票、ETF 和加密货币持仓
- 查看成本基础、盈亏情况和投资组合分配
- 将负债与资产一起跟踪
- 通过自动本地快照查看净资产历史
- 使用缓存的欧洲央行参考汇率转换数值
- 隐私模式下即时隐藏敏感金额
- 使用 Face ID 或 Touch ID 保护应用
- 随时导出、导入和删除本地数据

您的财务记录将保留在您的设备上。Wealth Compass 不要求注册账号，也不会将您的投资组合或交易历史存储在开发者运营的服务器上。

可选的市场价格更新使用您提供的 API 凭证。凭证会安全存储在 iOS 密钥串中。

Wealth Compass 是一个信息追踪工具。它不会连接金融机构、执行交易、提供财务建议，也不保证第三方市场数据的准确性。
