Wealth Compass คือแดชบอร์ดการเงินส่วนบุคคลสำหรับผู้ใช้งานในพื้นที่ท้องถิ่นแบบส่วนตัว ซึ่งช่วยให้คุณเข้าใจมูลค่าสุทธิ กระแสเงินสด การลงทุน สินทรัพย์คริปโต และหนี้สินของคุณได้ในที่เดียว

สร้างภาพที่ชัดเจนของการเงินของคุณ:

- ดูมูลค่าสุทธิ ยอดเงินสด การลงทุน และคริปโตได้ในทันที
- ติดตามรายได้และค่าใช้จ่ายด้วยหมวดหมู่และแนวโน้มทางสายตา
- กำหนดธุรกรรมที่เกิดซ้ำและรับการเตือนในพื้นที่ตามต้องการ
- ตรวจสอบหุ้น กองทุน ETF และการถือครองคริปโตเคอร์เรนซี
- ตรวจสอบต้นทุนกำไรขาดทุนและการจัดสรรพอร์ตโฟลิโอ
- ติดตามหนี้สินควบคู่กับสินทรัพย์ของคุณ
- ติดตามประวัติของมูลค่าสุทธิด้วยภาพสแนปชอตอัตโนมัติในพื้นที่
- แปลงค่าเงินโดยใช้ค่าอัตราแลกเปลี่ยนอ้างอิง ECB ที่เก็บไว้
- ปิดบังตัวเลขที่ละเอียดอ่อนได้ทันทีด้วยโหมดความเป็นส่วนตัว
- ปกป้องแอปด้วย Face ID หรือ Touch ID
- ส่งออก นำเข้า และลบข้อมูลในพื้นที่ของคุณเมื่อใดก็ได้ตามต้องการ

บันทึกทางการเงินของคุณจะอยู่บนอุปกรณ์ของคุณ Wealth Compass ไม่จำเป็นต้องมีบัญชีและไม่จัดเก็บพอร์ตโฟลิโอหรือประวัติการทำธุรกรรมของคุณบนเซิร์ฟเวอร์ของผู้พัฒนา

การอัปเดตราคาตลาดเป็นทางเลือกใช้ข้อมูลรับรอง API ที่คุณให้ ข้อมูลรับรองจะถูกเก็บอย่างปลอดภัยใน iOS Keychain

Wealth Compass เป็นเครื่องมือติดตามข้อมูล ไม่เชื่อมต่อกับสถาบันการเงิน ไม่ดำเนินการซื้อขาย ให้คำแนะนำทางการเงิน หรือรับประกันความถูกต้องของข้อมูลตลาดจากบุคคลที่สาม
