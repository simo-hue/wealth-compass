Wealth Compass 是一款私人、本地優先的個人理財儀表板，幫助你在一個地方了解你的淨資產、現金流、投資、加密資產和負債。

構建清晰的財務概況：

- 一目了然地查看淨資產、現金餘額、投資和加密資產
- 使用分類和視覺化趨勢追蹤收入和支出
- 排程定期交易並接收可選的本地提醒
- 監控股票、ETF 和加密貨幣持倉
- 查看成本基礎、盈虧及投資組合配置
- 與資產一起追蹤負債
- 自動本地快照追蹤淨資產歷史
- 使用緩存的 ECB 參考匯率轉換數值
- 使用隱私模式立即隱藏敏感金額
- 使用 Face ID 或 Touch ID 保護應用程式
- 隨時匯出、匯入和刪除你的本地數據

你的財務紀錄保留在你的設備上。Wealth Compass 不需要帳戶，也不會將你的投資組合或交易歷史存儲在開發者管理的伺服器上。

可選的市場價格更新使用你提供的 API 憑證。憑證安全地存儲在 iOS 金鑰圈中。

Wealth Compass 是一個資訊追蹤工具。它不連接金融機構、不執行交易、不提供財務建議，也不保證第三方市場數據的準確性。
